# 文案复制器

这是一个静态 HTML 文案复制工具，适合部署到 GitHub Pages。

## 文件说明

- `index.html`：网页主文件，已经内置文案复制功能
- `.nojekyll`：让 GitHub Pages 按静态文件直接发布

## GitHub Pages 部署

1. 新建一个 GitHub 仓库，例如：`caption-copy-tool`
2. 上传本文件夹里的 `index.html` 和 `.nojekyll`
3. 进入仓库 Settings → Pages
4. Source 选择 Deploy from a branch
5. Branch 选择 main，Folder 选择 /root
6. 保存，等待 1-3 分钟
7. 打开 GitHub Pages 提供的网址

## 手机使用

用 iPhone Safari 打开 GitHub Pages 链接，点“复制下一条文案”即可。
也可以在 Safari 中点击分享按钮，选择“添加到主屏幕”。
